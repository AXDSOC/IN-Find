<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/axdsys/IN-Find
```

<p>2. Cd IN-Find</p>

```
cd IN-Find
```

<p>3. Install requirements.txt</p>

```
pip install -r requirements.txt
```

<p>4. Run the python script</p>

```
python3 In_find.py
```
